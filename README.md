- 📡 **Backend:** API Routes in Next.js
- 🗄 **Database:** MongoDB (with Prisma ORM)
- 🔐 **Authentication:** NextAuth.js (Google, GitHub, and Email authentication)
- ⚡ **Deployment:** Vercel
